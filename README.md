# Poja Shri Portfolio

A modern and responsive personal portfolio website built using **Next.js** and **Tailwind CSS** to showcase my projects, technical skills, certifications, and achievements.

---

## Features

*  Modern UI with gradient and blob effects
*  Fully responsive design for desktop and mobile
*  Smooth animations and interactive sections
*  Dynamic certifications showcase
*  Resume download functionality
*  Stylish and clean portfolio layout
*  Animated components using Tailwind CSS

---

##  Tech Stack

* Next.js

* Git & GitHub
